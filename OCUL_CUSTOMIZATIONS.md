# OCUL Customizations

This file outlines the customizations that are included in this code base.

### OCUL Location Items Filter
This element simply expands the location item filters by default when viewing the full display of a record that contains multiple items (Under the Locations heading).

This customelement simply toggles the inner `filtersVisible` state of the host component to expand expand it by default.

[Learn more about this customization in the CFDUX spotdocs](https://docs.scholarsportal.info/view/Main/OCUL/CF/Committees/CFDUX/CFDUX_Configuration_Documentation/12_-_JS_changes/).
